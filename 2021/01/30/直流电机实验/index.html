直流他励电动机实验

MG：直流电动机DJ15
CG：涡流测功机
R1：选用D44的180 Ω电阻
Rf1：选用D44的1800Ω电阻
A1：直流毫安表，200mA档
A2：直流安培表

  1. 按图接线。测功机CG与电动机M同轴连接，作为直流电动机M的负载，用于测量电动机的转速、转矩和输出功率。
  2. 将直流电动机M的磁场调节电阻Rf1调至最小值，电枢串联起动电阻R1调至最大值，调节电枢电源的电压为220V。先接通励磁电源，观察到励磁电流If1后，接通电枢电源开关，使其起动，其旋转方向应符合转速表正向旋转的要求。
  3. M起动正常后，将其电枢串联电阻R1调至零，调节测功机D55-4挂件“给定调节”旋钮，先逆时针调至最小，再按调零“按钮”数秒，然后顺时针调节“给定调节”旋钮，同时观察电流表，使直流他励电动机的电枢电流逐渐上升至额定电流，使电动机达到额定值：$U=U_N=220 V，I=I_N = 1.2A ，n=n_N = 1600r/min$ ，此时MG的励磁电流If即为额定励磁电流IfN。
  4. 保持$U=U_N，If=I_{fN}$，逐次减小电动机负载（减小 “给定调节”旋钮），测取电动机电枢输入电流Ia，转速n和转矩T2及输出功率，共取数9-10组。

注意事项：

  1. a直流电动机M起动时，要注意须将R1调到最大，Rf1调到最小，先接通励磁电源，观察到励磁电流If1后，接通电枢电源，M起动运转。起动完毕，应将R1调到最小。
  2. 电枢电源、励磁电源、直流电流表正负极不能接反。
  3. 调节磁场调节电阻Rf1对电动机进行调速，电阻增大，转速增加。
  4. 直流他励电动机停机时，必须先切断电枢电源，然后断开励磁电源。同时必须将电枢串联的起动电阻R1调回到最大值，励磁回路串联的电阻Rf1调回到最小值，使电机MG停机。

直流并励发电机实验

  1. 在断电的条件下将发电机G的励磁方式接为并励，接线如图所示。Rf2调至最大阻值，负载电阻R2调到最大阻值，打开开关S。

  2. 起动电动机，调节电动机的转速，使发电机的转速n=nN，观察直流电压表V是否有剩磁电压，若无剩磁电压，可将并励绕组改接成他励方式进行充磁。

  3. 合上开关S逐渐减小Rf2 ，观察发电机电枢两端的电压，若电压逐渐上升，说明满足自励条件。如果不能自励建压，将励磁回路的两个端头对调联接即可。

  4. 调节电动机的磁场调节电阻Rf1（调转速）、发电机的磁场调节电阻Rf2（调端电压）和负载电阻R2 （调负载电流），使发电机的转速、输出电压和电流三者均达额定值，即n=nN = 1600，U=UN = 220V，IL=IN = 0.5A。

  5. 保持此时Rf2的值和n=nN不变，减小负载电阻R2 ，直至IL=0（断开开关S），从额定到空载运行范围内测取发电机的电压U和电流IL。读取6-7组数据。

  1. 找额定点即电压U=200V、电流I=0.5A、转速n=1600均达到额定值。
  2. 以电流为准合理采集数据，额定点附近应密集些，且额定点必测。
  3. 直流电动机MG起动时，要注意须将R1调到最大，Rf1调到最小，先接通励磁电源，观察到励磁电流If1为最大后，接通电枢电源，MG起动运转。起动完毕，应将R1调到最小。
  4. 调节负载电阻时同时关注发电机转速，保证转速为额定1600r/min。负载电流先增大后减小。

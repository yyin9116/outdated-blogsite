    建立传递函数相关的函数

假设系统是单输入单输出（SISO）系统，其输入、输出分别用$u(t)、y(t)$来表示，则得到线性系统的传递函数模型：

$G(s) = \frac{Y(s)}{U(s)}= \frac{b_ms^m+b_{m-1}s^{m-1}+\cdots+b_1s+b_0}{s^n+a_{n-1}s^{n-1}+\cdots+a_1s+a_0}$

在MATLAB中，可以利用传递函数分子、分母多项式的系数向量进行描述，分子num、分母den多项式的系数向量分别为：

$num=[b_m, b_{m-1},\cdots,b_0], den = [1, a_{n-1},\cdots,a_0]$

系数按$s$的降幂排列

  1. 建立传递函数模型的函数$tf()$

  2. 提取模型中分子分母多项式系数的函数$tf(data)$

    建立零极点形式的数学模型相关函数

$G(s) =K\frac{(s-z_1)(s-z_2)\cdots(s-z_m)}{(s-p_1)(s-p_2)\cdots(s-p_n)}$

[z]、[p]、[k]分别为系统的零极点和增益向量

  1. 建立零极点形式数学模型的函数$zpk()$

  2. 提取函数模型部分分式展开的函数$residue()$

    部分分式展开后，余数回到向量r, 极点回到列向量p， 常数项回到k。

    可以将部分分式转化为多项式比$p(s)/q(s)$。

MATLAB时域分析

时域分析函数：

  1. 单位阶跃响应函数step()

    • y = step(num, den, t)

    • [y, x, t] = step(num, den)

    • [y, x, t] = step(A, B, C, D, iu)

           以上函数将绘制波形图并求出其数值解，若只想绘制系统的阶跃响应曲线，则可采用step(num, den)、step(num, den, t)、step(A, B, C, D, iu, t)和step(A, B, C, D, iu)的格式进行函数调用。

     线性系统的稳态值可以通过控制系统的函数dcgain()求得

  2. 单位脉冲响应函数impulse()

  3. 零输入响应函数initial()

    • initial(sys, x0)和initial(sys, x0, t)
    • [Y, T, X] = initial(sys, x0)和[Y, T, X] = initial(sys, x0, t)

  4. 任意输入响应函数Isim()

    • Isim(sys1, u, t)和Isim(sys2, u, t, x0)

    • [Y, T, X] = Isim(sys1, u, t)和[Y, T, X] = Isim(sys2, u, t, x0)

      离散系统只需在连续系统对应函数前加‘d’即可
